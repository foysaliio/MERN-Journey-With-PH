1. [attr] - has attribute

2. [attr="value"] - exact match

3. [attr~="value"] - word in space-separated list

tag only

tag + featured

4. [attr|="value"] - exact or starts with "value-"

lang="en"

lang="en-US"

5. [attr^="value"] - starts with

https link

tel: link

6. [attr$="value"] - ends with

report.pdf

files.zip

7. [attr*="value"] - contains substring

contains "example"

8. Case-insensitive [attr="value" i]

data-status="Active"
data-status="ACTIVE"

9. Combined attribute selectors

10. Attribute + class combined

demo-item with active status
